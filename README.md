# Vue test-app

Задание взято с https://github.com/vad23klev/test-app-docker

Для работы проекта нужен работающий сервер с откликом на http://localhost:80/api/comments из вне docker контейнера

### Функциональные требования:

- добавление / удаление комментариев (без перезагрузки страницы)
- постраничный просмотр комментариев (3 комментария на страницу)
- сортировка по id и дате добавления в обе стороны
- сортировку и пагинацию необходимо реализовать на фронте
- для реализации использовать готовый пакет vue2-date-picker, установится при настройке проекта, для реализации работы можно использовать axios

### Требование к разработке:

- использование адаптивной верстки
- использование js-фреймворка vue
- загрузка комментариев по ajax **был использован axios**
- использовать vuex и реализовать возможность менять сортировку на клиенте.
- Форма добавления комментария должна располагаться под уже добавленными комментариями.

### Сборка проекта

```
npm install
```

### Запуск

```
npm run serve
```

### Production сборка

```
npm run build
```

### Lints & fix

```
npm run lint
```
